### Getting Started

To run the To-do List Application on your local machine, follow these steps:

1. **Clone the Repository**: Start by cloning this repository to your local machine using `git clone`.

2. **Install Dependencies**: Navigate to the project directory and install the required dependencies using the following command:

```bash
pnpm install
```

### Environment Configuration

Create a .env file in the root directory of the project and configure your environment variables. An example .env file might look like this:

```
PORT=3000
MONGODB_URI=mongodb://localhost:27017/todo-app
```
Modify these values to match your preferred configuration.

### Running the Application 

Once the dependencies are installed and the environment is configured, you can start the application using the following command:

```
pnpm start
```

This command will launch both the Express.js server and any other necessary processes concurrently.

### Accessing the API Documentation

After successfully starting the application, you can access the API documentation generated by Swagger by visiting http://localhost:3000/api/v1/docs/#/ in your web browser.
